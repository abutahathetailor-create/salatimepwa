# Salah Times - Jubail

A beautiful, responsive web app that displays real-time prayer times for Jubail, Saudi Arabia.

## Features
- Real-time prayer times for Jubail
- Countdown to next prayer
- Beautiful responsive design
- Works on desktop and mobile
- Offline fallback support

## How to Use
1. Open `index.html` in any web browser
2. Or visit the GitHub Pages URL

## Prayer Calculation Method
Uses University of Islamic Sciences, Karachi method

## Location
Jubail, Saudi Arabia (27.0040° N, 49.6460° E)
